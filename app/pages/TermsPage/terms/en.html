Preamble

Pitchy provides you with a test tool whose objective is to ensure the compatibility of your computer and your working environment with the Pitchy application.

If the test raises alerts, we invite you to contact Support (support@pitchy.fr), or your account manager if you benefit from specific support.

Items tested by the application on your browser:
  • Network port availability 80
  • Access to various AWS services used by Pitchy (Amazon Web Services)
  • Access to the Mixpanel service
  • Access to the Zendesk service
  • Pitchy JavaScript code execution
  • Browser type & version
  • Browser hardware acceleration
  • Browser IndexDB availability
  • Network speed and latency


Data kept for the purpose of improving the Pitchy service
  • User contact
  • Test date
  • Browser type & version
  • Network speed
  • Possible alerts raised by the test


Confidentiality of data

Pitchy agrees not to store any information that is not necessary for the purpose of continuous service improvement. At the express request of the user: Pitchy undertakes to delete all the information collected through this test.
